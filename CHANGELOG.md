# Changelog

All notable public milestones for this portfolio are documented here.

This project follows Semantic Versioning.

## [1.0.0] - 2026-05-15

### Added

- Added RafaelOS interactive portfolio experience.
- Added professional profile, experience, project, skills, resume and contact sections.
- Added SEO metadata, canonical URL, sitemap and robots configuration.
- Added structured data schemas for portfolio indexing.
- Added Vercel Analytics and Speed Insights.
- Added Google Analytics support through `NEXT_PUBLIC_GA_ID`.
- Added source-available license notice and asset permission tracking.
- Added Conventional Commits, commitlint, Husky and lint-staged workflow.
- Added GitHub Actions validation for typecheck, lint and production build.

### Changed

- Organized repository documentation for professional review.
- Documented branch, commit, pull request and release conventions.

### Fixed

- Improved publication workflow to avoid vague commit history.
